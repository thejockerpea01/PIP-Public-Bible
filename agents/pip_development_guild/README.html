Future Work Rules

Follow these rules before creating or editing any future PIP Development Guild visual.

Rules
  • Start from guild_dashboard.html for overview.
  • Use guild_layout.html for Guild Hall placement.
  • Use composition_bible.html for role integrity.
  • Use guild_bible.html for character identity.
  • Use cropped_sprites/front_cutouts/ for composites.
  • Do not redraw approved characters.
  • Do not use old full character sheet panels.
  • Do not modify approved PNGs unless explicitly requested.
  • Position edits must be micro patches only.
