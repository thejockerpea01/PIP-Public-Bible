Starter Prompt: เริ่มสร้างทีมใหม่

Create a new Visual Agent Team system.

Goal:
Build an interactive NPC-style team dashboard where each Agent is represented as a character in a shared scene.

Rules:
- Use a clean background as the base scene.
- Use transparent cutout sprites for characters.
- Do not use full character sheet panels as sprites.
- Store all role mapping in the Bible files.
- Place each Agent using one shared coordinate object.
- Sprite position, hitbox, hover label, and highlight ring must derive from the same coordinate data.
- Hover shows only name label.
- Click opens NPC-style dialogue.
- Dialogue includes:
  - ดูหน้าที่ของ Agent นี้
  - สั่งงาน Agent นี้
- Workflow mode shows checklist and copyable prompt template.
- Create archive/ for old drafts.
- Do not modify approved assets unless explicitly requested.
